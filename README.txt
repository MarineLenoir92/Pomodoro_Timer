Pomodoro Timer 

- Premier Projet Individuel

- Langages choisis: HTML, CSS, JavaScript

- Description Projet: Créer un countdown permettant d'appliquer la méthode Pomodoro dans mon quotidien. Ce projet est amener à évoluer en même temps que les connaissances acquises pendant ma formation. 

- Objectif : Parvenir à structurer un projet de A à Z. Travailler par itération selon la méthode Agile. Manipuler le CSS pour créer une interface stylisée. Travailler les fonctions asynchrones en JS. Créer des liens entre HTML, CSS et JS. 


Etape 1:

- Graphisme et Construction de la structure de la page web


Etape 2: 

- Mise en place de la base HTML


Etape 3: 

- Coder la Fonctionnalité 1 - Timer fonctionnel avec bouton start en 6 itérations

Itération 1
- [x]  Lorsque je suis sur la page du Pomodoro Timer: je dois voir le timer affiché à 25:00.
- [x]  Alors le timer se met en marche seul. Arrivé à 00:00, le timer doit s’arrêter (et ne pas descendre en dessous de 0)


Itération 2
- [x]  Lorsque je suis sur la page du Pomodoro Timer: je dois voir le timer affiché à 25:00 ainsi qu’un bouton start cliquable.
- [x]  Une fois le bouton cliqué alors le timer se met en marche. Arrivé à 00:00, le timer doit s’arrêter (et ne pas descendre en dessous de 0)


Itération 3
Lorsque je suis sur la page du Pomodoro Timer: 
- [x]  je dois voir le timer affiché à 25:00 dans un cadre stylisé (cf Etape 1)
- [x]  un bouton start cliquable, stylisé (cf Etape 1) et changer de couleur lorsque la souris le survole avec un pointeur en forme de main.


Itération 4
Lorsque le timer arrive à 00:00 alors le compteur s’arrête et:
- [x]  l’utilisateur doit également être avertit par un son de la fin du compte à rebours.


Itération 5
Lorsque je suis sur la page du Pomodoro Timer:
- [x]  Le body doit être 50% en une couleur 50% d’une autre couleur.


Itération 6
Lorsque le timer se met en route :
- [x]  Le timer doit se déclencher en parallèle dans l’onglet
- [x]  Un message “UNE PAUSE S’IMPOSE!” doit s’afficher lorsque le timer s’arrête
- [x]  Ajouter une image dans l’onglet de tomate ou réveil
